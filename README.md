# PolyTools 🛠️

PolyTools is a comprehensive web application suite built with Streamlit, offering various utility tools for different needs. This all-in-one application combines several practical features in a simple and intuitive user interface.

## 🌟 Features

### 📍 Map Visualizer
- Canadian postal code visualization on an interactive map
- Support for GPS coordinates and city names
- Intuitive user interface with Folium map integration
- Performance-optimized SQLite database

### ⏲️ Timer Manager
- Create and manage multiple timers
- Clear and responsive user interface
- Automatic timer state saving
- Visual alerts for expired timers

### 📊 Inventory Manager
- Complete inventory management system
- SQLite database for data storage
- Update and consultation interface
- Multiple configurable storage locations

### 🔄 Multi-Converter
- Multiple conversion types:
  - Volume
  - Distance
  - Surface
  - Morse Code
  - Special Characters (Alpha)
  - Various Codes

### 📝 List Tools
- List manipulation tools
- Postal code generator
- Intuitive data management interface

### 🏥 Pillow Manager (Oreiller.py)
- Pillow tracking system
- Multilingual support
- Date and comment management
- Data editing interface

## 🚀 Installation

1. Clone the repository:
```bash
git clone https://github.com/PolyG0ne/PolyTools.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Launch the application:
```bash
streamlit run Home.py
```

## 📦 Main Dependencies

- streamlit
- folium
- streamlit-folium
- pandas
- sqlite3
- plotly

## 🔧 Configuration

The application uses several configuration files:
- SQLite database for inventory
- JSON files for state saving
- CSV files for postal codes

## 🌐 Usage

1. Launch the application using `streamlit run Home.py`
2. Access the interface through your browser (default: http://localhost:8501)
3. Navigate between different tools using the main menu
